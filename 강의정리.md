### 1. Swift 입문

- 변수 : 필요에 따라 저장된 값을 여러 번 다른 값으로 변경할 수 있다. 
- 상수 : 한 번 저장된 값을 다른 값으로 변경할 수 있다.
- 타입
  - Int : 정수값을 저장하는 자료형
  - String : 문자열을 저장하는 자료형
  - Bool : 참(True) 또는 거짓(False)를 저장하는 자료형
  - Double : 실수값(소수점)을 저장하는 자료형
- 주석
  - // : 한줄 주석
  - /* content */ : 여러줄 주석
--- 
### 2. 변수와 상수의 선언과 초기화
``` swift
// 닉네임, 나이, 이메일, 평점, 성별
/*
Practice1. SignUser
*/

var nickname: String = "kch"
var age: Int = 23
let email: String = "adasddasd12@naver.com" // 상수(let)
var rate: Double = 4.5
var gender: Bool = true // true: 남자, false: 여자

print("안녕하세요 \(nickname)입니다. 저는 \(age)살이고, 이메일은 \(email)입니다. 평점은 \(rate)에요")
// \(변수명) : "패키징" 이라고 부름
```

### 3. 변수, 상수의 차이
``` swift 
// 닉네임, 나이, 이메일, 평점, 성별
/*
Practice1. SignUser
*/

var nickname: String = "kch"
var age: Int = 23
let email: String = "adasddasd12@naver.com" // 상수(let)
var rate: Double = 4.5
var gender: Bool = true // true: 남자, false: 여자

print("안녕하세요 \(nickname)입니다. 저는 \(age)살이고, 이메일은 \(email)입니다. 평점은 \(rate)에요")
// \(변수명) : "패키징" 이라고 부름

+++++++++++++++++++++++++++++++++++++++++++++++++추가된 코드+++++++++++++++++++++++++++++++++++++++++++++++++

// 6개월이 지나 회원정보 수정

age = 20
rate = 3.3
gender = false
// email = "test@test.com"  // email은 상수(let)으로 선언 됐으므로 변경 불가

print("안녕하세요 \(nickname)입니다. 저는 \(age)살이고, 이메일은 \(email)입니다. 평점은 \(rate)에요")
```

### 4. 선언과 초기화, 자료형 정리
- 선언 : ex) var name: String
- 초기화 : ex) name = "kch"
- 상수(let)에 저장된 값은 변경될 수 없음
- 변수 타입에 일치하지 않는 값이 저장될 시 에러 발생
  - ex) var age: String = "23"
- 데이터 값이 명시적인 경우, 타입을 생략할 수 있음(타입 추론)
  - ex) var name = "kch"
- 데이터 타입이 명시해둔 경우(타입 어노테이션)
  - ex) var name: String = "kch"

### 5. 변수와 상수의 활용
``` swift
/*
Practice2.WhoAmI

- \n: Escape Sequence
*/

let userBirth = 1999  // camelCase 각 단어의 첫 글자를 대문자로 표기한다.
let day = (2021 - userBirth) * 365
let rice = day * 3

print("""
저는\n태어난 지 100개월로,
\(day)일째 살고 있습니다.
지금까지 \(rice)끼를 먹었고,
\(day * 8)시간을 잤어요.
""")
```
